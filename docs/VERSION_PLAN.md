# ApksCC 版本规划文档

## 项目信息
- **仓库**: https://github.com/Shunamxiao/SplitAPKInstaller
- **技术栈**: React Native 0.73 + TypeScript
- **包名**: com.apkscc

---

## v1.0.0 (2026-02-13)
### 功能
- ✅ 支持 APK/XAPK/APKS/APKM 格式安装
- ✅ 自动解压和解析安装包
- ✅ 自动复制 OBB 数据文件
- ✅ 使用 Android PackageInstaller API 安装 Split APKs

### 已知问题
- 权限请求不完善

---

## v1.0.1 (2026-02-13)
### 修复
- ✅ 添加 Android 11+ 特殊权限请求模块
- ✅ 启动时检测权限状态
- ✅ 界面显示权限授权按钮
- ✅ 点击后跳转系统设置页面

## v2.0.0 (开发中)
### 产品定位
从单一安装器 → 游戏资源盒子（安装器作为核心功能之一）

### 改动
- 应用改名: Split APK Installer → **ApksCC**
- 包名改为: com.apkscc

### 新功能

#### 首页 (Home)
- [ ] 顶部轮播图（Banner）
- [ ] 游戏分类标签（热门/新游/破解/汉化）
- [ ] 游戏卡片列表（封面+名称+大小+下载按钮）
- [ ] 搜索入口

#### 我的 (Profile)
- [ ] 用户头像/昵称区域
- [ ] 功能入口列表：
  - 📦 本地安装器（原安装功能）
  - 📥 下载管理
  - ⭐ 我的收藏
  - ⚙️ 设置

#### 文件关联
- [ ] 注册为 .xapk/.apks/.apkm 的打开方式
- [ ] 从文件管理器点击直接调起安装

### 技术方案
- 底部 Tab 导航（首页/我的）
- React Navigation
- 模拟数据（后续接 API）

### UI 参考
- TapTap
- 酷安
- 各大游戏盒子
